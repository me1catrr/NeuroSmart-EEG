Publicacion y alcance

Esta web publica una portada en docs/ y centraliza el acceso a los modulos HTML ubicados en docs/Pluto/.

  • Los notebooks fuente viven en Pluto/ y se mantienen como codigo cientifico.
  • Las exportaciones HTML de Pluto.jl se preparan fuera de docs/ y luego se sincronizan para publicar.
  • Esta portada funciona como punto de acceso unificado a los modulos exportados.
Sitio publicado con GitHub Pages desde la carpeta docs.
